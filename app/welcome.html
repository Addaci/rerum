Welcome to RERUM!

This is an open and free repository for all sorts of digital things. Choose an option below to learn more or register your application.

1. API 2. register a server 3. discover rerum objects 4. available rerum tools 5. about us / contact

In this virtual place you will find many things wondrous to behold. Digital anchors for real world objects and encoded assertions from real world people are stored here without prejudice.

At the moment, anything that can be represented as an oa:Annotation, an ore:Aggregation, or several types of IIIF objects can find a permanent home here. More options will be added as this place of things matures.
